# LearnDrack – AI-Powered Learning Platform

## Overview

LearnDrack is a full-stack AI-powered learning platform designed to help users create, manage, and explore educational content across various domains. The platform provides an interactive learning environment with course management, lesson organization, user authentication, and AI-assisted learning features powered by Google Gemini AI.

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/LearnDrack.git
cd LearnDrack
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_secret_key
```

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
